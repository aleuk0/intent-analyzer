На основе диалогов строится дерево разговора.

Что происходит:
1. Прочитываются файлы и разговоры сохраняются в промежуточную переменную
2. Разговоры обрабатываются, определяется намерение, разговоры сохраняются в удобной для представления графа форме
3. Строится граф, исключая циклические ссылки, для упрощения одно намерение используется один раз
4. Результат выводится стандартным выводом и сохраняется в файл result.json
